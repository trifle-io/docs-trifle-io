# Changelog

### **2.0.0** - *March 2, 2026*

  `Trifle.Stats` 1.0.0 was published to the plugin repository in 2023. Since then, no additional versions have been released there up until now.

  Development has continued actively in the GitHub repository. The project reached feature parity with its Ruby counterpart in late 2025 and has continued to track its capabilities. It is actively used in the Trifle App and remains production-tested.

`Trifle.Stats` hasn't been released to plugin repository. That doesn't mean it hasnt been maintained or updated. It reached feature parity with its Ruby cousin sometime in late 2025 and from there it kept matching its features. It's been actively used in Trifle App.
